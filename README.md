# CivilizationVI_CommunityExtension

This mod is currently in beta. Report bugs on the [issue tracker](https://github.com/Wild-W/CivilizationVI_CommunityExtension/issues). You can also contact me directly or on the steam workshop page. I prefer if you do it on the repo though.

[Visit the wiki!](https://github.com/Wild-W/CivilizationVI_CommunityExtension/wiki)

## FAQ

### What does this mod do?
This mod exposes new tools for modders to use by editing Civ VI's base code.
